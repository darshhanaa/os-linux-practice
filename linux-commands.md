# Linux Commands

## System Information

- hostname → shows system name
- who → shows logged-in users
- whoami → current user
- pwd → current directory
- date → system date and time
- cal → calendar

---

## Navigation

- ls → list files
- ls -l → detailed list
- ls -a → hidden files
- cd folder → change directory
- cd .. → go back

---

## Terminal Control

- clear → clear screen
- history → show command history
- exit → close terminal
